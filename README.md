## Description

</p>

## Installation

### Requirements

- Node.js >= 14.0
- npm >= 6.0

### Steps

1. **Clone the repository:**

   ```bash
   git clone https://github.com/vinn-as1999/buzzchat
   cd buzzchat


2. **Install the dependencies:**
```npm install```

3. **Usage:**
```npm run dev```
